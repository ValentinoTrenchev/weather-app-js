About the App

Weather App is a simple and intuitive application that provides real-time weather information for cities around the world.

Features include:

  • Current weather statistics (temperature, humidity)
  • Hourly weather forecasts
  • Warmest and coldest times prediction
  • Search for any city worldwide

Data is provided by the OpenWeatherMap API.

About the Creator

  • Vanilla JavaScript (ES6+)
